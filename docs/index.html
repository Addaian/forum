AI Jury

How each top finding got its verdict

  • Six debaters, each named after the one value they care about and indifferent to the other five: ● Simplicity · ● Velocity · ● Maintainability · ● Correctness · ● Scalability · ● Flexibility
  • 15 pairs per finding — every combination of the 6 values, (Simplicity vs Velocity, Correctness vs Velocity, etc.). Each pair has a brief 2-turn exchange; a neutral observer then records what the pair concluded.
  • Each persona argues whatever its value reads in the evidence — debt OR justified. Convergence is a strong signal; divergence gets resolved by the vote-extractor.
  • The judge reads all debates and renders one verdict. May override the majority when the evidence supports it.

              How do I run the server?
            
            
# 1. install (one time)
uv pip install -e ".[server]"

# 2. set secrets
export ANTHROPIC_API_KEY=sk-...
export FORUM_API_TOKEN=$(openssl rand -hex 16)   # share this with your friends

# 3. boot
uv run uvicorn server:app --port 8001

# 4. expose to the internet (pick one)
cloudflared tunnel --url http://localhost:8001   # zero account, random URL
ngrok http 8001                                  # stable URL with free account

# 5. paste the tunnel URL + token into this dialog
